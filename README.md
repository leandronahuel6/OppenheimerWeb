# Oppenheimer Web Project 🎬⚛️

Un sitio web interactivo, completamente responsive y optimizado, diseñado como tributo cinematográfico a la película *Oppenheimer*. 

Este proyecto fue desarrollado como parte de mi formación en la **Tecnicatura Universitaria en Programación** en la **UTN Facultad Regional Haedo**, con el objetivo de aplicar las mejores prácticas del desarrollo frontend moderno, ademas de que es una trabajo practico para la materia Programacion 4.

## 🚀 Tecnologías y Características
- **HTML5 & CSS3 Semántico:** Arquitectura limpia, uso de CSS Variables, Flexbox y Grid.
- **JavaScript (ES6):** Interactividad con DOM, `IntersectionObserver` para contadores numéricos y lógica de Easter Eggs multimedia.
- **Formulario Funcional:** Integración con la API de Formspree para recepción de correos reales.
- **Diseño UI/UX:** Efectos de *glassmorphism*, animaciones fluidas, protección de tablas responsive y página de Error 404 personalizada.
- **SEO & Performance:** Metaetiquetas Open Graph (Redes Sociales) y optimización de imágenes (WebP/AVIF).
